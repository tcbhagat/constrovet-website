# Constrovet SEO/GEO Launch Checklist

Use this after the SEO pages are deployed to `https://www.constrovet.com`.

## 1. Verify Deployment

- Open `https://www.constrovet.com/`.
- Confirm these return `200`:
  - `https://www.constrovet.com/robots.txt`
  - `https://www.constrovet.com/sitemap.xml`
  - `https://www.constrovet.com/pages/construction-cost-leakage-audit.html`
  - `https://www.constrovet.com/pages/construction-cost-overrun-analysis.html`
  - `https://www.constrovet.com/pages/construction-esg-carbon-audit.html`
  - `https://www.constrovet.com/pages/schedule-slippage-recovery.html`
  - `https://www.constrovet.com/pages/construction-due-diligence-financiers.html`

## 2. Google Search Console

- Add or open the property for `https://www.constrovet.com/`.
- Submit sitemap URL: `https://www.constrovet.com/sitemap.xml`.
- Use URL Inspection for:
  - `https://www.constrovet.com/`
  - `https://www.constrovet.com/pages/construction-cost-leakage-audit.html`
  - `https://www.constrovet.com/pages/construction-due-diligence-financiers.html`
- Request indexing only after each page returns a clean live test.

## 3. Bing Webmaster Tools

- Add `https://www.constrovet.com/` or import from Google Search Console.
- Submit sitemap URL: `https://www.constrovet.com/sitemap.xml`.
- Run URL Inspection and Site Scan for the same URLs listed above.
- Optional later step: configure IndexNow only after standard sitemap submission is working.

## 4. Structured Data And Performance

- Test important pages in Google Rich Results Test.
- Test important pages in PageSpeed Insights.
- Keep images, metadata, and schema evidence-safe. Do not add fake ratings, reviews, client logos, client names, awards, or performance numbers.

## 5. Weekly Tracking

Track these queries in Search Console and Bing Webmaster Tools:

- `construction cost leakage audit software India`
- `construction cost overrun analysis India`
- `construction ESG carbon audit construction`
- `schedule slippage recovery construction project`
- `construction project due diligence financiers`
- `cost leakage audit for construction projects`

Record every week:

- Indexed pages count.
- Impressions by query.
- Clicks by page.
- Average position for the target queries.
- Pages with crawl or indexing errors.
- Evidence received for any future claim, client proof, certification, metric, or testimonial before publishing it.
