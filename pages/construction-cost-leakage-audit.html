What It Finds

Cost leakage means evidenced value loss such as penalties, wastage, rework, delay cost, idle resources, delayed PO impact, excess consumption, or calculated overruns. BOQ value, planned spend, contract value, work done, and advance recovery stay baseline budget.

SignalCategoryReason
BOQ valueBASELINE_BUDGETApproved baseline for comparison.
Rework cost with source noteLEAKAGE_AND_OVERRUNPotential value loss when evidence supports it.
Actual greater than BudgetLEAKAGE_AND_OVERRUNCalculated as Actual - Budget when both values are available.

Budget vs Leakage

Cost leakage is not total project value, BOQ value, planned spend, work done, or advance recovery. Constrovet keeps those baseline values separate.

Leakage findings are created only when project evidence supports them.

What Documents Help

  • Budget, actual spend, BOQ, contract value, and work-done statements.
  • Procurement records, purchase orders, invoices, and material consumption logs.
  • Daily progress reports, delay notes, site instructions, and schedule updates.
  • ESG, energy, fuel, water, carbon, waste, and emissions records when available.

What Executives Get

Each finding should show the issue, category, amount if quantified, days if evidenced, citation, confidence, and missing evidence.

FAQ

Does every budget variance become leakage?

No. Baseline budget stays baseline. Leakage needs supporting evidence.

How are overruns calculated?

When Budget and Actual are both available, overrun is Actual - Budget.

Can the system cite evidence?

Yes. Each finding should include file, page or sheet, and quoted span.
